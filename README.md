# Graphing Calculator

![Graph](ReadMe.png)


## About
Graphing Calculator using the Kivy Framework! 

## Features

- Create any equation using the sidebar

- Realtime Updating, used for when moving the graph

- Graph is Resizable to any window size

- Custom Equation Solver
